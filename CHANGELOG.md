# Change Log

All notable changes to the "guml" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [0.2.0] - 2026-04-02

### Added

- Syntax highlighting (TextMate grammar + semantic tokens) for `.guml` files
- Code completion for components, properties, events, global references, and enum values
- Hover information with type details and event signatures
- Go to Definition for imports and parameter declarations
- Full-document and range formatting
- Real-time syntax and semantic diagnostics
- Document highlight for symbol occurrences
- Built-in snippets for common GUML patterns
- Incremental parsing support (configurable via `guml.incrementalParsing`)
- Quick Actions command for fast access to analyzer operations
